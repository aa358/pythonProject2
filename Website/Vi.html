What is Vi?

Vi is a visual text editor that comes with Unix operating systems by default. It's a character-based screen editor with no graphical user interface. Vi's ubiquity and accessibility are two of its most appealing features. It may be used without a desktop environment, and it is usually already installed on your computer.

Vi Commands


  • vi filename - edit filename starting line 1
  • :q (Return) - quit or exit vi
  • j or (down arrow key) - move cursor down one line
  • u - Undo
  • i - insert text before cursor
  • r - replace single character under cursor
  • :r filename (Return) - read file named filename and insert after current line
  • :w (Return) - write current contents to file named in original vi call
  • n - move to next occurrence of search string
