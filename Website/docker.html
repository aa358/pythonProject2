What is Docker?

Docker is a series of platform-as-a-service technologies that provide software in containers using OS-level virtualization. Containers are separated from one another, with their own software, libraries, and configuration files, and can only communicate with one another over well-defined channels.

More About Docker

Containers are “runnable instances of an image. You can create, start, stop, move, or delete a container using the Docker API or CLI. You can connect a container to one or more networks, attach storage to it, or even create a new image based on its current state.”
